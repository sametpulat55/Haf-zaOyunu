# Hafıza Oyunu 
Atatürk Üniversitesi Yazılım Mühendisliği projelerim kapsamında geliştirdiğim bir hafıza kartı oyunu.

##  Özellikler
- Python ve Gradle tabanlı yapı.
- Görsel eşleştirme mantığına dayalı oyun mekaniği.
- Otomatik görsel oluşturma desteği (`generate_images.py`).

##  Kullanılan Teknolojiler
- Python
- Gradle / Java
- Git
